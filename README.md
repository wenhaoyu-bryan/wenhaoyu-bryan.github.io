# Wenhao Yu — AI PM

Personal brand website for Wenhao Yu (Bryan), an AI Product Manager focused on agentic workflows, ontology-driven systems, vibe coding, and SEO/GEO growth experiments.

**Live site:** https://wenhaoyu-bryan.github.io

## Pages

- **Home** — Hero, featured projects, topic cards, latest writing, contact
- **Projects** — Case studies including Prompt-to-Ontology
- **Playbook** — AI PM Playbook with 5 categories and 4 core skills
- **Writing** — Blog posts on AI PM, agents, ontology, vibe coding, SEO/GEO
- **Growth Lab** — SEO/GEO experiments and growth systems
- **About** — Narrative intro, background, tool stack, contact

## Tech Stack

- [Astro](https://astro.build) — Static site generator
- [AstroPaper](https://github.com/satnaing/astro-paper) — Base theme (heavily customized)
- [Tailwind CSS](https://tailwindcss.com) — Styling
- [Pagefind](https://pagefind.app) — Static search
- GitHub Pages — Hosting via GitHub Actions

## Local Development

```bash
pnpm install
pnpm run dev       # http://localhost:4321
pnpm run build     # production build → dist/
```

## Deployment

Push to `main` triggers the GitHub Actions deploy workflow automatically.
